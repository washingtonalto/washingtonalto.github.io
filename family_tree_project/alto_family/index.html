üöÄ How to Contribute

üìå Note: This is a collaborative project! Family members are encouraged to add or update information.
  1. View and edit the source Excel file for the family tree via the link above
  2. Add or update family member information via Data Entry Tab:
    • PersonID (unique number)
    • Name
    • Nickname (optional)
    • Gender (male/female)
    • Birth/Death dates (optional)
    • Father, Mother, Spouse (references to other Persons via PersonIDs)
  3. Save your changes
  4. Submit via:
    • Email to the family tree maintainer
    • GitHub pull request (if you're tech-savvy)
    • Share via family group chat

Usage
