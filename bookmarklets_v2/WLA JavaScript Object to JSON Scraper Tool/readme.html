WLA JavaScript Object to JSON Scraper Tool

This bookmarklet allows you to inspect any JavaScript object on a webpage and extract it as a structured JSON file. Once executed on the current page, it prompts you to select a global object and then downloads its key/value pairs as a JSON file.

üìå What It Does

  • Prompts user to enter or select an in-page JavaScript object.
  • Traverses the object properties and serializes them into JSON.
  • Generates a downloadable JSON file of the object structure.

üì• Installation Instructions

To use this tool, you need to save it as a bookmark in your browser.

  1. Open your browser‚Äôs bookmark manager.
  2. Create a new bookmark and give it a title like WLA Object to JSON Scraper.
  3. Copy the bookmarklet code (below) and paste it into the bookmark‚Äôs URL field.
  4. Save the bookmark.

üõ† How to Use

  1. Navigate to any web page where you want to extract a JavaScript object.
  2. Click the ‚ÄúWLA Object to JSON Scraper‚Äù bookmarklet.
  3. Follow the on-screen prompt to enter the object name (e.g., document.location).
  4. The script will serialize its properties and prompt you to save a .json file.

‚öôÔ∏è Notes & Tips

  • The bookmarklet runs in the context of the current page and has access to its global JavaScript variables.
  • Only enumerable properties will be extracted ‚Äî functions and unsupported types may be skipped or represented differently.
  • This tool is useful for debugging, scraping in-page objects, or exporting dynamic data from single-page apps without opening DevTools.
