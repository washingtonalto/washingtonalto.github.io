WLA Tealium Check Page Tool

This bookmarklet audits a webpage for common Tealium data layer values and related tracking objects (e.g., `utag_data`, `utag.data`). It displays key/value pairs for analysis and debugging.

üìå What It Does

  • Extracts relevant Tealium variables available in the page context.
  • Displays them in a readable table format.
  • Useful for QA or debugging tag management setups.

üì• Installation

  1. Open your browser‚Äôs bookmarks manager.
  2. Add a new bookmark called ‚ÄúWLA Tealium Check‚Äù.
  3. Copy and paste the JS bookmarklet (below) into the URL field.
  4. Save and execute on any page you want to audit.
